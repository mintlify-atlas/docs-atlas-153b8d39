    <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick start</h2>
    <p className="text-base text-gray-600 dark:text-gray-400 max-w-2xl mx-auto">
      Get up and running with Manifest in minutes
    </p>
  </div>

  <Steps>
    <Step title="Install the plugin">
      Install Manifest as an OpenClaw plugin with a single command:

      ```bash
      openclaw plugins install manifest
      ```

      The plugin hooks into your OpenClaw gateway and starts routing and tracking all LLM requests automatically.
    </Step>

    <Step title="Choose your mode">
      Manifest runs in two modes — cloud (default) or local:

      <Tabs>
        <Tab title="Cloud mode">
          Sign up at [app.manifest.build](https://app.manifest.build) to get your API key, then configure:

          ```bash
          openclaw config set plugins.entries.manifest.config.apiKey "mnfst_YOUR_KEY"
          openclaw gateway restart
          ```

          Your telemetry flows to the cloud dashboard with secure blind proxy architecture.
        </Tab>
        <Tab title="Local mode">
          Run the embedded server locally with zero external dependencies:

          ```bash
          openclaw config set plugins.entries.manifest.config.mode local
          openclaw gateway restart
          ```

          Dashboard opens at **http://127.0.0.1:2099**. All data stays on your machine.
        </Tab>
      </Tabs>
    </Step>

    <Step title="View your dashboard">
      Your agent telemetry starts flowing immediately. Open the dashboard to see:

      - Real-time cost and token tracking
      - Message logs with full conversation history
      - Model usage breakdowns across 300+ supported models
      - Smart routing decisions and scoring insights

      <Note>
        In local mode, the dashboard shows an orange **Dev** badge. In cloud mode, access your dashboard at app.manifest.build.
      </Note>
    </Step>
  </Steps>
</div>
