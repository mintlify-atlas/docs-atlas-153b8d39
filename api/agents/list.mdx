---
title: List Agents
api: GET /api/v1/agents
---

Retrieves all active agents for the authenticated user, along with their usage statistics and activity sparklines.

## Response

<ResponseField name="agents" type="array">
  Array of agent objects with usage statistics

  <Expandable title="Agent Object">
    <ResponseField name="agent_name" type="string" required>
      The agent's unique slug identifier
    </ResponseField>

    <ResponseField name="display_name" type="string" required>
      Human-readable display name for the agent
    </ResponseField>

    <ResponseField name="message_count" type="number" required>
      Total number of messages processed by this agent
    </ResponseField>

    <ResponseField name="last_active" type="string" required>
      ISO 8601 timestamp of the agent's last activity
    </ResponseField>

    <ResponseField name="total_cost" type="number" required>
      Total cost in USD for all messages processed by this agent
    </ResponseField>

    <ResponseField name="total_tokens" type="number" required>
      Total token count (input + output) for all messages
    </ResponseField>

    <ResponseField name="sparkline" type="array" required>
      Array of hourly token counts for the last 7 days (downsampled to 24 points)
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash
curl https://api.manifest.build/api/v1/agents \
  -H "Cookie: better-auth.session_token=YOUR_SESSION_TOKEN"
```

## Notes

- Agents are ordered by creation date (newest first)
- Only active agents are returned
- Sparklines show hourly token usage over the last 7 days, downsampled to 24 data points
- Statistics are calculated across all messages for each agent
